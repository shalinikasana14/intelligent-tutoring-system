# Intelligent Tutoring System — RAG Pipeline & LLM Evaluation

An agentic NLP system built using Retrieval-Augmented Generation (RAG) 
to deliver adaptive, context-aware responses through LLM-based reasoning.

## Tech Stack
Python | LangChain | FAISS | Hugging Face Transformers | PyTorch | TensorFlow | NumPy | Pandas

## Overview
- Designed and built an end-to-end data pipeline covering dataset extraction, 
  curation, annotation and quality validation.
- Built a RAG architecture combining FAISS-based semantic retrieval with 
  LangChain and Transformer-based LLMs for context-aware output generation.
- Designed an automated evaluation framework to track system outputs, 
  measure response quality across varied inputs and drive iterative improvement.
- Integrated structured numerical data with unstructured text retrieval 
  within a single end-to-end pipeline.
